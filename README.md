# POS (Point Of Sale)

## Feature
- Create a new sale
- Select all sale

## Installation
 1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd pos
   ```
   2. Install the required installeents [the only external dependency you'll need!]
   ```bash
   go mod tidy
   ```
   3. Run application
   ```bash
   go run cmd/main.go
   ```
